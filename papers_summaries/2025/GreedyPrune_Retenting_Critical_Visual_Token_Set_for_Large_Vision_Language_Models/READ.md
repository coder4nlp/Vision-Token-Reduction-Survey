# 研究背景
**​​研究问题​**​：这篇文章要解决的问题是大型视觉语言模型（LVLMs）在图像理解任务中计算效率低的问题，特别是在资源受限的设备上。由于处理大量视觉令牌的高成本，LVLMs的部署面临显著挑战。

**​​研究难点​**​：该问题的研究难点包括：现有的基于语义显著性和视觉多样性的剪枝方法各有局限，前者可能忽略视觉多样性，后者可能在高压缩比下丢弃语义重要的令牌。
